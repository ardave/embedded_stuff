# Project: ESP32 Blink LED (C++)

## Build System
- ESP-IDF framework
- Before building, run: `get_idf` (alias for `. $HOME/esp/esp-idf/export.sh`)

## Common Commands
- `idf.py build` - Build the project
- `idf.py flash` - Flash to device
- `idf.py monitor` - Serial monitor
- `idf.py flash monitor` - Flash and open monitor
